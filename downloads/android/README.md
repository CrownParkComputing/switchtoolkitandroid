## üöÄ Installation Instructions

### Step 1: Download APK
- Download the latest APK from the table above
- **Recommended**: Use the latest version (v2.0.1)

### Step 2: Enable Unknown Sources
1. Open **Settings** on your Android device
2. Navigate to **Security** (or **Apps & notifications** > **Advanced** > **Special app access**)
3. Find **Install unknown apps** or **Unknown sources**
4. Select your browser or file manager
5. **Enable** "Allow from this source"

### Step 3: Install APK
1. Open your Downloads folder
2. Tap on `SwitchToolkit-v2.0.1.apk`
3. Tap **Install**
4. Wait for installation to complete
5. Tap **Open** or find SwitchToolkit in your app drawer

### Step 4: Initial Setup
1. **Grant Permissions**: When prompted, allow storage permissions
2. **Library Setup**: Choose your game library folder
3. **Wait for Scan**: Initial library scan may take a few minutes
4. **Ready to Use**: Start managing your game collection!

## üêõ Troubleshooting

### Installation Issues
- **"App not installed"**: Ensure Unknown Sources is enabled for your file manager
- **"Parse error"**: Download the APK again, file may be corrupted
- **"Insufficient storage"**: Free up at least 100MB space

### Runtime Issues
- **Library not loading**: Grant storage permissions and refresh library
- **Emulator won't launch**: Verify emulator is installed and assigned correctly
- **NSZ decompression fails**: Check prod.keys are imported and valid

### Performance Tips
- **Large libraries**: Use fast storage (internal storage preferred over SD card)
- **Memory usage**: Close other apps when processing large batches
- **Battery optimization**: Disable for SwitchToolkit to prevent background interruptions
